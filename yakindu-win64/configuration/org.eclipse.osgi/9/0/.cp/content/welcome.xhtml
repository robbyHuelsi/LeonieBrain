Getting started

YAKINDU Statechart Tools (SCT) is an integrated modeling environment for the specification and development of reactive, event-driven systems based on the concept of statecharts. It is an easy-to-use tool featuring sophisticated graphical statechart editing, validation and simulation of statecharts as well as generation state machine code for C, C++, and Java.
